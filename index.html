WEBアプリケーションの開発から実行、運用までのすべてをクラウドで完結できるクラウドサービス。

クラウドサービスは大きくSaaS、PaaS、IaaSの3種類に分けられ、Herokuはこのうち、PaaS（Platform as a Service）に分類されます。

本来だったら自分でサーバやOS、DBなどを準備しなければならないが、Herokuにはそれらが全て揃っているので、手軽にアプリを公開することができます。

MERIT

デプロイが簡単

デプロイは基本的にHerokuにpushするだけです。デプロイ作業に時間がかかって大変！ということはありません。

多様な言語・FWをサポートしている

現在では国内で扱われている主要言語とFW（以下）をほぼカバーしています。

  • Ruby(Rails)
  • PHP
  • Java
  • Node.js
  • Python
  • Go

Gitコマンドで操作できる

Herokuではアプリケーションをデプロイするには、Gitコマンドで可能です。 Herokuをリモートリポジトリとしてgit pushするだけで手軽にアプリケーションを公開することができます。
